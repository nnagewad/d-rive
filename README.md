# Dérive

A Dérive is a French concept meaning an aimless walk through city streets, that follows the whim of the moment. It is sometimes translated as a drift. ([Laboratory Landscape](https://www.sfu.ca/~sbitter/Laboratory_Landscape/derives.html#:~:text=A%20D%C3%A9rive%20is%20a%20French,sometimes%20translated%20as%20a%20drift.))
